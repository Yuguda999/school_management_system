## â“ Troubleshooting

### Upload Fails

**Problem**: File won't upload
**Solutions**:
- Check file size (max 50MB)
- Verify file type is supported
- Check your storage quota
- Try a different browser

### Preview Not Working

**Problem**: Can't preview material
**Solutions**:
- Only PDF, images, and some documents support preview
- Try downloading instead
- Check browser compatibility
- Update your browser

### Can't Find Material

**Problem**: Material doesn't appear in list
**Solutions**:
- Check if filters are applied
- Verify material is published (not draft)
- Use search function
- Check if material was deleted

### Sharing Issues

**Problem**: Students can't access shared material
**Solutions**:
- Verify material is published
- Check share permissions
- Ensure expiration date hasn't passed
- Confirm student is in the selected class
